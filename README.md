PROJECT TOTURIAL: https://rogueliketutorials.com/

| Key press | Description               |
| --------- | ------------------------- |
| ↑         | move up                   |
| ↓         | move down                 |
| ←         | move left                 |
| →         | move right                |
| g         | pick up Items             |
| i         | open the inventory        |
| d         | drop item from inventory  |
| c         | show the character status |
| /         | look around               |
| >         | go to next floor          |
| v         | view the log history      |
| esc       | exit and save the game    |
